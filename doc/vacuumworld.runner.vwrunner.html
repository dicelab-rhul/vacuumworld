class VWRunner(multiprocessing.context.Process)
VWRunner(config: dict[str, typing.Any], minds: dict[vacuumworld.common.vwcolour.VWColour, vacuumworld.model.actor.mind.surrogate.vwactor_mind_surrogate.VWActorMindSurrogate], allowed_args: dict[str, typing.Type[typing.Any]], **kwargs: Any) -> None
 
This abstract class is the base for all the VacuumWorld runners. It contains all the common methods and attributes.
 
All the runners must implement the `run()` method.
 
All the arguments passed to `VacuumWorld.run()`are stored and validated here, together with their default values.
