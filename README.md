# vacuumworld

### Install

```
pip install vacuumworld 
```

### Install from source: 

``` 
git clone git@github.com:dicelab-rhul/vacuumworld.git 
cd vacuumworld
pip install -e .
```

**NOTE:** Some additional dependancies may be required for linux users:
```sudo apt-get install python3-pil python3-pil.imagetk```
